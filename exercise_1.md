# P5704 【深基2.例6】字母转换

## 题目描述

输入一个小写字母，输出其对应的大写字母。例如输入 q[回车] 时，会输出 Q。

## 输入格式

无

## 输出格式

无

## 输入输出样例 #1

### 输入 #1

```
q
```

### 输出 #1

```c
Q
```



## 我的解答

``````c
#include <stdio.h>

int main()
{
    char alph;
    scanf("%c",&alph);
    printf("%c",alph - 32);
    return 0;
}
``````
